Question 1

[1013599] X plc has non-current assets £200k, trade receivables £50k, creditors £30k, overdraft £20k and inventory £30k. What are X plc's shareholders' funds? You should answer this question by typing the correct numbers in the following format without symbols: 000,000
The correct answer is: 230,000 - 230,000

Explanation

Shareholder's funds are the value left in the company after debt has been paid, i.e. net assets. Non-current assets + current assets - liabilities = shareholders' funds Receivable and inventory are current assets. Creditors and overdraft are liabilities. £200k + £50k + £30k - £30k - 20k = £230k

Question 10

[1013653] Given the following cash flow information, what is the operating profit? Operating cash flow - £10m; depreciation - £2m; increase in inventory - £3m; increase in payables - £4m.
A
£2m
B
£5m
C
£7m
D
£9m
The correct answer is: C - £7m

Explanation

This requires the reconciliation adjustments to be made. This question wants to reconcile the profit with the given cash flow. Generally, we reconcile the cash flow with the given profit, using the equation below: Operating profit + depreciation - increase in inventory + increase in payables = net cash flow from operating activities To assess the profit, we rearrange the equation: Operating profit = Net cash flow - depreciation + increase in inventory - increase in payables = £10m - £2m + £3m - £4m = £7m.

This requires the reconciliation adjustments to be made. However, in this question it asks us to recocile the profit with the net cash flow. Operating profit = £75,000 - £40,000 - £15,000 + £10,000 Operating profit = £30,000
